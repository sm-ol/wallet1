Как запустить программу:

Откройте терминал, введите команду: python cli.py

Описание:
Программа имитирует работу кошелька. Поддерживает различные операции: пополнение, списание, создание кошелька. Все денежные расчеты ведутся в точном типе Decimal для защиты от потери копеек, а интерфейс полностью отделен от бизнес-логики.
Реализована на классах Wallet, ExchangeAccount и WalletError.

Описание (Макет с классами):
Класс Wallet: 
- Метод parse_money: переводит ввод в Decimal и округляет до 2 знаков после запятой
- Метод top_up: осуществляет пополнение кошелька и возвращает новый баланс
- Метод spend: осуществляет списание и возвращает новый баланс

Класс ExchangeAccount:
- Метод init: создает словарь для хранения уже созданных кошельков
- Метод validate_wallet_name: проверяет имя на пустоту и очищает пробелы
- Метод create_rub_wallet: проверяет, можно ли создать кошелек с такими данными, и если да, добавляет его
- Метод get_wallet: находит нужный кошелек в словаре

Класс WalletError: работа с ошибками бизнес-логикиТестирование: проект покрыт тестами с использованием "pytest"Для их запуска в терминале введите команду: pytest test_wallet.py -v
